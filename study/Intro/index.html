JavaScript Intro

  • 웹 퍼블리싱 작업은 전체적으로 html, css, javascript 언어를 사용하여 이루어집니다.
  • 먼저 html 언어를 통해 검색 창, 버튼, 이미지 등과 같은 웹의 가장 기초적인 뼈대를 작성하고, css 언어로 웹을 디자인하며, javascript 언어를 이용하여 웹의 동작을 구현할 수 있습니다.
  • javascript는 타입을 명시할 필요가 없는 인터프리터 언어이며 객체지향 언어입니다. 이는 주로 웹 브라우저에서 사용되나, Node.js와 같은 프레임워크를 사용하면 서버측 프로그래밍에서도 사용이 가능합니다.
  • 기본적으로 자바스크립트는 html 문서 위에서 동작하는 언어입니다. html 문서에 자바스크립트를 직접적으로 접목시키는 방법은 html 파일에 태그를 사용하는 것입니다. 그러면 이 태그의 안쪽에 위치한 코드를 자바스크립트의 코드로 해석하게 됩니다.
  • 자바스크립트를 이용하면 html 코드의 요소의 속성값을 손쉽게 변경할 수 있습니다. 즉, 자바스크립트의 목적은 사용자와 상호작용하는 웹 페이지를 만들기 위한 것입니다.
